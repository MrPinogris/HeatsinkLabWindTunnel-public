How to import into Excel
1
Open Excel, go to the Data tab and click From Text/CSV.
2
Navigate to the logs folder inside the program directory.
3
Choose the CSV file you want. Sort by Date modified to find the newest one.
4
Click Import.
5
A preview window appears — click Transform Data. Do not click Load.
6
Power Query opens — click Advanced Editor in the top-left ribbon.
7
Select everything after the Source = Csv.Document(...) line and replace it with the formula in the dialog.
8
The editor should look like this. Click Done, close Power Query, and click Keep when prompted.
